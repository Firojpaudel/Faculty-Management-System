# Faculty Management System

A comprehensive desktop-based management system for academic faculties, built with Java, MySQL, and Embedded Tomcat.

## Features
- **User Authentication**: Role-based access control for Admins, Staff, and Students.
- **Student Dashboard**: View attendance, subjects, materials, and exam results.
- **Administrative Tools**: Manage users, students, staff, and notices.
- **Automated Database Initialization**: Smart schema execution and data seeding.

## Tech Stack
- **Language**: Java 17
- **Database**: MySQL 8.0 (XAMPP compatible)
- **Web Server**: Embedded Apache Tomcat 10
- **Build Tool**: Maven
- **UI Framework**: Java Swing with FlatLaf
- **Libraries**: Jackson (JSON), MySQL Connector/J
